01 · 项目概述

用大模型重新定义磁盘清理

DiskMind 是面向 Windows / macOS 双端的桌面级智能磁盘分析与清理工具。 本地扫描采集元数据 → LLM 语义判定可清理性 → 自然语言解释 + 风险评分， 把"看不懂、不敢删"变成"看得懂、敢放心删"。

🔍

规则 + 语义双引擎

传统规则库覆盖已知文件，LLM 推断长尾未知文件的归属与用途。

🧠

风险评分 + 自然语言

每条建议附 0–100 风险分与一句话解释，用户决策有理有据。

🛡️

沙箱清理 · 可回滚

所有删除先入应用 Trash，30 天可一键恢复，杜绝"误删神话"。

🌐

跨平台一致体验

Tauri 2.0 + Rust 内核，Mac/Win 同一套交互与品牌心智。

🔒

隐私优先模式

本地 Ollama 推理可选,文件内容永不外发,仅元数据可见。

💬

对话式问答

"这个 6GB 文件能删吗?" 直接问 AI,得到上下文相关的回答。

06 · AI 应用

差异化的核心:LLM 真正落地

LLM 不是宣传词。结构化输出 + 元数据隐私 + 双轨推理,让 AI 成为产品本身。

四个 LLM 真正在做的事

① 长尾文件性质判定

规则库无法识别的 `~/.config/random-tool/cache.db`,LLM 基于路径与上下文推断归属。

② 风险评分 + 自然语言解释

每条建议附 0–100 风险分 + 一句话理由,普通用户读得懂、信得过。

③ 对话式问答

"这个 Xcode/DerivedData 能删吗?" — AI 上下文回答,而非搜索结果。

④ 个性化策略学习

用户保留/删除/恢复行为持续微调建议权重,做到越用越懂你。

07 · 风险分析

直面风险:每一条都有解

清理工具最大的敌人是误删事故,LLM 最大的敌人是幻觉。两者叠加,需要工程上严防死守。

误删核心文件高

直接造成系统/数据损坏,触发"灾难级"口碑反噬。

缓解: 沙箱清理 + 白名单 + 30 天回滚 + 高风险二次确认
LLM 幻觉中

模型给出错误的删除建议,误导用户。

缓解: JSON Schema 强约束 + 二次规则校验 + 证据不足返回 unknown
隐私顾虑中

用户担心文件元数据外泄到云端 LLM。

缓解: 本地模式 + 上传字段透明审计 + 可视化上传清单
大厂入场中

微软/苹果系统内置类似 AI 功能。

缓解: 跨平台一致体验 + AI 深度领先 + 品牌叙事
扫描性能低

大磁盘扫描超过 5 分钟,首次体验差。

缓解: 增量扫描 + 后台 watcher + 进度可视化
LLM 成本失控低

API 涨价或被滥用,毛利收窄。

缓解: 多供应商可切换 + 缓存复用 + 本地兜底
